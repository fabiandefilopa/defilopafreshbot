# SOL FINDER

> 🇪🇸 **¿Preferís leer en español?** → [Ver documentación en español](es/)

**On-chain intelligence for Solana wallets.**

<figure><img src=".gitbook/assets/banner.png" alt=""><figcaption></figcaption></figure>

SOLFINDER is a Telegram bot that helps you track and analyze wallet activity on the Solana blockchain. It combines two powerful tools:

* **Fresh Wallet Scanner** — Detect brand-new wallets receiving funds from major exchanges.
* **Privacy Cash Scanner** — Trace funds flowing through Privacy Cash, Solana's mixing protocol.

Whether you're researching smart-money movements or investigating obfuscated transactions, SOLFINDER gives you actionable on-chain data in seconds — right from Telegram.
