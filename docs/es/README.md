# SOLFINDER

**Inteligencia on-chain para wallets de Solana.**

SOLFINDER es un bot de Telegram que te ayuda a rastrear y analizar la actividad de wallets en la blockchain de Solana. Combina dos herramientas:

- **Fresh Wallet Scanner** — Detecta wallets nuevas que reciben fondos desde exchanges.
- **Privacy Cash Scanner** — Rastrea fondos que pasan por Privacy Cash, el protocolo de mixing de Solana.

Ya sea que estés investigando movimientos de smart money o rastreando transacciones ofuscadas, SOLFINDER te da datos on-chain en segundos, directo desde Telegram.
