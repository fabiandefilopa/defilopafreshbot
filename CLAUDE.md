# Project Rules

## Git workflow
- Always push to `origin main` immediately after every commit. Never leave commits unpushed.
- Railway auto-deploys from GitHub on every push to main.

## Language
- Respond in Spanish unless the user asks otherwise.

## Bot
- Bot runs on Railway via polling (not webhook).
- Bot username: @solfinder_app_bot
- All user-facing text in the bot is in English.
